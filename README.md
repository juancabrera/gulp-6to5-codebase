Gulp babel codebase
==================
A basic [gulp](https://github.com/gulpjs/gulp) codebase project using [babelify](https://github.com/babel/babelify) and [browserify](https://www.npmjs.com/package/browserify) for its modules.

### Getting started
1. `npm install`
2. `gulp`
